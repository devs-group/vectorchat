VectorChat Neon Widget (Standalone)

Run locally with Vite:

1) cd 2
2) npm install
3) npm run dev
4) Open the shown local URL (defaults to http://localhost:5173)

The demo page at index.html includes the widget via a simple script tag:

<script src="/vectorchat-neon-widget.js"></script>

Build/preview:
- npm run build
- npm run preview
